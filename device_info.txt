# 1. How an ML Model is Actually Trained (End-to-End)

## Step 1: Data Collection

You need **relevant, labeled data**.

**Sources:**

* Public datasets (Kaggle, UCI)
* APIs (Twitter, weather, etc.)
* Your own app logs / database
* Sensors / scanners (like your EID project)

**Example (your use case):**

* Input: Image / barcode / text
* Output: Extracted ID (label)

---

## Step 2: Data Preprocessing

Raw data is useless without cleaning.

**Tasks:**

* Remove noise / missing values
* Normalize (scale numbers)
* Encode categorical data
* Image resizing (for CV)
* Tokenization (for NLP)

**Python stack:**

```python
import pandas as pd
from sklearn.preprocessing import StandardScaler
```

---

## Step 3: Train-Test Split

Avoid overfitting.

```python
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
```

---

## Step 4: Model Selection

Depends on problem type:

| Problem        | Models                          |
| -------------- | ------------------------------- |
| Classification | Logistic, RandomForest, XGBoost |
| Regression     | Linear, Ridge, Lasso            |
| Image          | CNN (TensorFlow/PyTorch)        |
| Text           | Transformers                    |

---

## Step 5: Training

Model learns patterns.

```python
from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier()
model.fit(X_train, y_train)
```

---

## Step 6: Evaluation

Check performance.

```python
from sklearn.metrics import accuracy_score

pred = model.predict(X_test)
print(accuracy_score(y_test, pred))
```

---

## Step 7: Save Model

You don’t retrain every time.

```python
import joblib
joblib.dump(model, "model.pkl")
```

# 2. How Model is Deployed (This is where most beginners struggle)

You **DON’T embed ML directly in frontend**.

You wrap it inside a backend API.

---

## Step 1: Create Backend API

Use:

* Flask (simple)
* FastAPI (recommended)

### Example:

```python
from fastapi import FastAPI
import joblib

app = FastAPI()
model = joblib.load("model.pkl")

@app.post("/predict")
def predict(data: list):
    result = model.predict([data])
    return {"prediction": result.tolist()}
```

---

## Step 2: Run Server

```bash
uvicorn app:app --reload
```

Now your ML model is live at:

```
http://localhost:8000/predict
```

---

## Step 3: Connect Frontend (Website)

Frontend sends request to backend.

### Example (JavaScript):

```javascript
fetch("http://localhost:8000/predict", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify([1,2,3,4])
})
.then(res => res.json())
.then(data => console.log(data));
```
